# Init Sensei

This method initializes a brand new Sensei node.  It can only be called once during intial setup.  It is used to set your node's admin username and passphrase.

This method will also instantiate your admin token.  This means this endpoint is unauthenticated.


### Endpoint

POST `/api/v1/init`

### Authentication

No authentication required.

### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| username | String | Used for logging into the web administration panel |
| passphrase | String | Used to encrypt seed information on disk and as your password for logging into the web administration panel |

### Response Parameters

<br />

| Param | Type | Description |
| ----- | ---- | ----------- |
| token | String | admin `token` string |
