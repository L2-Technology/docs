# Start Sensei

This method starts a lightweight node. On success it will include a Set-Cookie header that contains your macaroon information.  This can be used to start and login to your node using one request. 

### Endpoint

POST `/api/v1/nodes/start`

### Authentication

This request is unauthenticated but requires a valid passphrase.


### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| pubkey | String | The node's public key. This is often described as the node's id |
| passphrase | String | Used to encrypt seed information on disk and as your password for logging into the web panel |

### Response Parameters

<br />

| Param | Type | Description |
| ----- | ---- | ----------- |
| macaroon | String | Hex encoded `admin.macaroon` file |
