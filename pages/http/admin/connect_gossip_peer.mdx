# Connect Gossip Peer



### Endpoint

POST `/api/v1/peers/connect`

### Authentication

You must authenticate using an admin token


### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| node_connection_string | String | Public key and address of target gossip peer "publickey@address" |


### Response Parameters

There are no response parameters.
