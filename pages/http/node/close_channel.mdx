# Close Channel

This method closes a channel.

### Endpoint

POST `/api/v1/node/channels/close`

### Authentication

You must authenticate using the admin node's macaroon.

### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| channel_id | String | ID of channel to close |
| force | Boolean | If the channel should be force-closed |

### Response Parameters

<br />

There are no response parameters.
