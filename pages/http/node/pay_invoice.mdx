# Pay Invoice

Make a payment from your node using a lightning invoice.

### Endpoint

POST `/api/v1/node/invoices/pay`

### Authentication

You must authenticate using the admin node's macaroon.

### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| invoice | String | Bolt11 lightning invoice |
