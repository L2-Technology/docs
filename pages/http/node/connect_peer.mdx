# Connect Peer

This method connects a new peer to our node. 

### Endpoint

POST `/api/v1/node/known-peers`

### Authentication

You must authenticate using the admin node's macaroon.

### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| label | String | A label for the peer |
| pubkey | String | The peer's public key |
| zero_conf | Boolean | Allow channel requests with zero confirmations |

### Response Parameters

<br />

There are no response parameters.
