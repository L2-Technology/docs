# Stop

Stop the node.

### Endpoint

POST `/api/v1/nodes/stop`

### Authentication

You must authenticate using the admin node's macaroon.

### Request Parameters

<br/>

| Param | Type | Description |
| ----- | ---- | ----------- |
| pubkey | String | The node's public key |

### Response Parameters

<br />

There are no response parameters from this request.
