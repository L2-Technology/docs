# Sensei

**Sensei** is an enterprise-grade lightning network node implementation built using the [Bitcoin Development Kit](https://bitcoindevkit.org) and the [Lightning Development Kit](https://lightningdevkit.org).

Sensei offers users multiple options to manage their node:  

- A web based admin interface that can be accessed from any browser.  
- A cli tool (senseicli) for issuing commands directly to the daemon. 
- An api that can be consumed over http or grpc.
